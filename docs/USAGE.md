# How to Use Autalon Generator

## Step 1: Generate the Katalon script

## Step 2: Download Autalon Driver

Download the latest from here: https://github.com/RoganMatrivski/AutalonDriver-Java/releases

## Step 3: Add driver to katalon project

Go to Menu Bar > Project > Settings. On side bar, select `Library Management`. Press `Add`, and then pick the driver file you've just downloaded.

## Step 4: Create new test case

## Step 5: Paste the script file you've generated

Open the test case in Katalon, go to the `Script` tab on the bottom, and then paste the script.
